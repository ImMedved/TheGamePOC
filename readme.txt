This version of the project already has a multithreaded implementation, as the project concept doesn't imply a sequential implementation. This project was almost completely finished in C++, but I rewrote it in Java in the last five days, so most of the functionality was cut to meet the deadline. This includes animation gameSystems, camera tracking, texture rendering for the level, characters, projectiles, the collision gameSystem, the HUD, and the menu. I may implement all of this in Java later. Regarding the key parts of the project, multiplayer works as a PTP packet exchange. I didn't have time to implement cheat checking, as mentioned in the project text, in this version. In any case, given the number of packets, which is 20 per second at best, this check would have been difficult to implement in the sequential version anyway. The Javadoc in Main.java describes the algorithm for launching on two devices on a local network.
